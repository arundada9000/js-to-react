## Prerequisites

- A modern web browser (Chrome, Firefox, Edge, Safari)
- No build tools, no npm, no frameworks required

## Running the App

1. Clone or download the repository:
   ```bash
   git clone https://github.com/arundada9000/js-to-react.git
   ```
2. Open `index.html` in your browser.
3. Start learning!

Or visit the live site: **[pre-mern.vercel.app](https://pre-mern.vercel.app)**

For local development with live reload, use [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) in VS Code (port 5501 recommended).

## Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `Ctrl + K` or `/` | Focus search |
| `[` | Collapse all phases |
| `]` | Expand all phases |
| `G` | Toggle grid view |
| `Ctrl + D` | Download curriculum as Markdown |
| `?` | Show all shortcuts |
| `Esc` | Close modals / clear search |
