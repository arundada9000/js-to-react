## About

This project is a **comprehensive JavaScript curriculum** built as a single-page app with vanilla HTML, CSS, and JavaScript. It covers 10 progressive phases from JavaScript basics to advanced concepts, preparing learners for React.

**Created by:** [Arun Neupane](https://arunneupane.netlify.app)  
**GitHub:** [@arundada9000](https://github.com/arundada9000)  
**Portfolio:** [arunneupane.netlify.app](https://arunneupane.netlify.app)

## Project Status

- ✅ Core curriculum with 10 phases
- ✅ Code runner with output terminal
- ✅ Flashcards study mode
- ✅ Pomodoro & Lo-Fi widgets
- ✅ Dark mode
- ✅ Progress tracking (localStorage)
- 🚧 PWA support *(coming soon)*

## Tech Stack

- **HTML5** — Semantic markup
- **CSS3** — Custom properties, Grid, Flexbox, animations
- **Vanilla JavaScript** — ES6+, modular architecture
- **Prism.js** — Syntax highlighting
- **canvas-confetti** — Completion celebrations
- **Font Awesome 6** — Icons
- **Google Fonts** — Inter & JetBrains Mono
